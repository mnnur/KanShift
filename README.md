## KanShift – Kanji Converter Extension for Chrome
![Kanshift](ks.png)
KanShift is a Chrome Extension that converts Japanese Kanji text into Hiragana, Katakana, or Romaji.
It’s powered by [Kuromoji](https://github.com/hexenq/kuroshiro-analyzer-kuromoji) and [Kuroshiro](https://github.com/hexenq/kuroshiro), running entirely offline.

## Features
- Convert Kanji to Hiragana, Katakana, or Romaji
- Right-click select “Convert Kanji” from any webpage text selection
- Works offline

##  Installation
### Option 1 – From Source (Unpacked Extension)

1. Clone or download this repository
2. Open Chrome and go to: chrome://extensions/
3. Enable Developer mode (toggle in top right)
4. Click Load unpacked and select the project folder
5. The KanShift icon should appear in your Chrome toolbar

## Usage
### Context Menu
1. Select any Japanese text on a webpage
2. Right click select Convert Kanji
3. A popup window will open showing the converted Kanji

### Manual Conversion
1. Click the KanShift icon in your toolbar
2. Paste or type Japanese text
3. Select conversion type and click Convert
